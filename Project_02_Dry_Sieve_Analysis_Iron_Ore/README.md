
# Dry Sieve Analysis for Iron Ore

## Project Overview

This project presents the laboratory investigation of **Dry Sieve Analysis for Iron Ore**, carried out as part of the Civil Engineering laboratory coursework at the **Indian Institute of Technology (Indian School of Mines), Dhanbad**. The experiment was conducted to determine the particle size distribution of an iron ore sample using a standard set of sieves.

The results help in understanding the grading characteristics of the material, which are important for mineral processing, aggregate analysis, and engineering applications.

---

## Objectives

- Determine the particle size distribution of an iron ore sample.
- Perform dry sieve analysis using standard laboratory sieves.
- Calculate the percentage retained and cumulative percentage passing.
- Plot the particle size distribution curve.
- Interpret the grading characteristics of the sample.

## Methodology

1. The iron ore sample was dried and weighed.
2. A stack of standard sieves was arranged in descending order of opening size.
3. The sample was placed on the top sieve and shaken for the specified duration.
4. The material retained on each sieve was collected and weighed.
5. Percentage retained, cumulative percentage retained, and percentage passing were calculated.
6. The particle size distribution curve was prepared using the obtained results.
